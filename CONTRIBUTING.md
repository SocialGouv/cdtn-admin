# Contributing

> Thanks for being willing to contribute !

## TL;DR

```sh
# Install
$ yarn

# Starts dev server
$ yarn start dev

# Build
$ yarn start build
# Watch the build
$ yarn start build --watch

# Lint
$ yarn start lint

# Test
$ yarn start test
# Watch the test
$ yarn start test --watch

# Build
$ yarn start e2e
# Watch the e2e
$ yarn start e2e --watch
```
